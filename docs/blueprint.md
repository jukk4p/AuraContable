# **App Name**: InvoiceFlow

## Core Features:

- User Authentication: User authentication with email and Google Sign-In.
- Dashboard Summary: Dashboard overview of issued, paid, and pending invoices.
- Invoice Management: CRUD operations for invoices, including details like invoice number, client, concepts, amounts, dates, status, and notes.
- Invoice Listing: Invoice listing with filtering and search functionality.
- Payment Status: Ability to mark invoices as paid.
- Invoice Export & Sharing: PDF Export and Email sending: Tool for creating invoice PDF ready to send and attach to e-mail based on invoice content.

## Style Guidelines:

- Primary color: HSL(210, 70%, 50%) which converts to a vibrant blue (#3399FF), evoking trust and clarity.
- Background color: HSL(210, 20%, 98%) which converts to a very light blue (#F0F8FF), for a clean and professional look.
- Accent color: HSL(180, 60%, 40%) which converts to a teal-ish turquoise (#33A6A6) to indicate actionable items.
- Body and headline font: 'Inter' (sans-serif) for a modern, neutral, and readable interface.
- Use simple, outline-style icons for clarity and a modern aesthetic.
- Clean, card-based layout for displaying invoice summaries and details.
- Subtle transitions and animations on user interactions to enhance the user experience.
